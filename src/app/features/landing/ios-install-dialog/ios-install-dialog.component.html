Instalar no iPhone ou iPad

No iOS, a instalação é feita diretamente pelo Safari. Siga estes passos:

  1. Abra este site no Safari.
  2. Toque no botão Partilhar (quadrado com seta para cima).
  3. Selecione Adicionar ao ecrã principal.
  4. Toque em Adicionar para concluir.

Depois de instalado, o SkillBridge fica disponível como aplicação no seu ecrã principal.
